# ImproveMAX Phase 1 Smoke Checklist

Use this checklist before and after each Phase 1 feature slice.

## Core CRUD

1. Create a new custom card.
2. Edit an existing custom card.
3. Delete a custom card.
4. Create a collection.
5. Edit a collection.
6. Delete a collection.
7. Create a workspace slot.
8. Edit a workspace slot.
9. Delete a workspace slot.

## Search And Organization

1. Search by title.
2. Search by tag.
3. Search by detail text.
4. Search by imported markdown body.
5. Search by collection name.
6. Search by workspace slot label after adding a card to a slot.
7. Toggle `Search body` on and off and confirm results change.
8. Switch sort mode and confirm ordering changes.
9. Pin a card and confirm it persists after refresh.
10. Favorite a card and confirm it persists after refresh.

## Relations

1. Add a relation from one card to another.
2. Change the relation type on the same pair.
3. Remove the relation.
4. Confirm the relation appears in detail view.
5. Confirm the relation appears in both graph views.

## Workspace And Reader

1. Add a card to the workspace from the card action.
2. Add a card to the workspace from the detail action.
3. Drag a card into a workspace slot.
4. Remove a card from a workspace slot.
5. Open the reader overlay from a card.
6. Open the reader overlay from the detail panel.

## Imports And Export

1. Import at least one markdown file.
2. Search imported markdown content.
3. Export a collection as Markdown.
4. Export a collection as JSON.
5. Export the workspace as Markdown.
6. Export the workspace as JSON.

## Views And Persistence

1. Switch between card view and graph view.
2. Select a resource and verify detail panel + local graph stay in sync.
3. Refresh and confirm:
   - selected resource restores
   - search input restores
   - pin/favorite restores
   - workspace contents restore
   - relations restore
4. Toggle zh/en and confirm new search and organization controls also translate.
