# 已知问题

当前版本是 `Windows Portable Beta`，请按 beta 心态使用。

## 已知限制

- 某些 Windows 外部拖拽场景下，文件类型识别可能不稳定
- 桌面壳首次启动可能比普通原生应用更慢
- 图谱视图在较老设备上仍可能出现局部掉帧
- 一部分低频中文文案仍在持续整理中

## 当前最稳的替代路径

- 导入文件优先使用 `导入文件` 按钮
- 如果桌面壳异常，先用 demo 版确认资源和图谱流程
- 如果图谱卡顿，优先关闭设置里的硬件加速做对比

## 不建议的使用方式

- 一次性拖入大量复杂 PDF 与图片混合资源
- 在桌面版异常时反复双击多个 exe 进程
- 把当前 beta 当作已经完全稳定的正式生产工具

## 反馈时最有价值的信息

- 你导入的文件类型
- 你用的是拖拽还是按钮导入
- 你是在 demo 版还是桌面版
- 异常出现在卡牌视图还是图谱视图

这些信息能显著加快问题定位速度。
