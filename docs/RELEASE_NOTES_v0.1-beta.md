# IPM v0.1-beta 发布说明

## 版本定位

这是一个可下载、可运行、可体验方向的 `Windows Portable Beta`。

它已经可以帮助你：

- 管理 AI 相关资源卡
- 导入本地资料
- 观察局部图谱
- 把核心资源编排进工作台

但它还不是正式版产品。

## 这一版重点

- 补齐了便携版打包链路
- 增加了 `导入文件` 按钮，降低 Windows 拖拽兼容性风险
- 持续收敛图谱视图的交互、层级与性能
- 修复了多个可见中文乱码与局部详情交互问题

## 发布建议

推荐在 GitHub 上以：

- `v0.1-beta`
- `Windows Portable ZIP`

的形式对外发布。

## 适合怎么发

不要把它包装成成熟 SaaS。

更适合的表达是：

> 这是一个本地优先的 AI 资源操作系统原型，用来沉淀提示词、知识卡、工作流和 Agent 规则，并通过图谱与工作台形成可复用的个人生产系统。
